Golden Rules (গোল্ডেন রুল)

Strict Prohibition

If information is missing in the pasted text, clearly state: "not clearly stated in provided text".

পেপারে যা নেই, সেটাকে “fact” বানাবে না। guess করবে না।

Implementation Logic

If you must assume a parameter for code execution, label it explicitly as an ASSUMPTION.

Dataset না পেলে substitute dataset নেবে এবং রিপোর্টে সেটা লিখবে।
