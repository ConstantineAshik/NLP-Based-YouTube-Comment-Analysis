Foundations of Classification

Understanding how models separate data in 1D and 2D space.

1. The 1D Threshold ("The Sliding Door")

In the simplest scenario (1 dimension), we only have one attribute to make a decision. The model learns a single number: the Threshold.

💡 Memory Anchor: The Sliding Door

Think of a sliding door. Anything to the left is "In", anything to the right is "Out". Moving the slider changes who gets in.

Accuracy: 0% Errors: 0

Key Insight: Notice how you can rarely get 100% accuracy if the blue and orange dots overlap (noise). This is reality.

2. The 2D Linear Model ("The Ruler")

With two attributes (X and Y), the decision boundary becomes a line. Linear classifiers (like basic SVM or Logistic Regression) try to find the "best fit" line to separate the classes.

💡 Memory Anchor: The Ruler

Imagine placing a rigid ruler on a table to separate apples from oranges. You can rotate it (slope) and slide it (intercept).

Accuracy: 0% Equation: y = mx + b

The Math: The model predicts Blue if the point is above the line, and Orange if below.
